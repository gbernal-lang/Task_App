Los CBV, son vistas, pero usando clases se caracterizan por ser organizadas, además de ser usadas para aplicaciones de gran tamaño.
Por otro lado las FBV, son vistas, pero en este caso usando funciones, estas son más sencillas de implementar estas se recomiendan usarse en vistas para aplicaciones pequeñas.
-------------------------------------------------------------------------------------------------------------------------------
Los mixins y herencia, van de la mano, ya que en los dos se pueden heredar funcionalidades de otras clases, sin embargo la diferencia, es que en la herencia se tiene que tener una relacion entre si, como "Carro" y "Vehículo", y con los mixins no se necesita tener esta relación.
